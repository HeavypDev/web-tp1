Projetos Realizados

Dieta Diária

Projeto desenvolvido para o curso Student to Business. Foi criado um app para Windows 8 para controle e organizaçao de uma dieta, de forma que o usuário possa acompanhar sua alimentação ao longo de um mês seguindo uma dieta previamente prescrita.

Melhoramento no algoritmo de comparação de proteínas

Como trabalho de conclusão de curso foi desenvolvido um método de comparação de proteínas para que possa ser usado como melhoramento dos algoritmos de sobreposição existentes.
